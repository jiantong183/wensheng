# 纹生 WenSheng · 中国传统纹样程序化生成器

> 算法生其形，人文赋其意。

「纹生」是一个**单文件、零依赖**的网页小程序，用纯几何与分形算法**程序化生成**中国传统纹样（回纹、冰裂纹、缠枝纹），并为每一种纹样配以人工撰写的文化释义卡片。

本项目是课程论文《审美主体的让渡与回归——论人工智能技术与中国传统艺术的融合、碰撞与变革》的实证案例，用以印证论文的核心观点：

> **AI 能高效复现传统艺术的「形式层」，意蕴层与文化层的「意义」却必须由人来赋予。**
> 机器精确地生成纹样的几何骨骼（对称、二方连续、分形），而母题的选择、象征的阐释、应用的判断，仍由人这一审美主体完成——此即「审美主体的回归」。

## 三种纹样的生成原理

| 纹样 | 几何规律 | 算法实现 |
|------|----------|----------|
| 回纹 | 二方连续 | 「回」字折线单元沿水平方向镜像、等距重复 |
| 冰裂纹 | 递归分形 | 矩形区域沿长边随机二分，逐层递归切分 |
| 缠枝纹 | 旋转对称 | 对数螺旋模拟藤蔓，绕中心旋转复制并缀以花苞 |

## 技术栈

- 原生 HTML + CSS + JavaScript（Canvas 2D）
- 无任何第三方框架与依赖
- 无任何预存图片，所有纹样均由代码实时生成
